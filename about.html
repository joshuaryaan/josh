Introduction

Josh Warner is a freelance web designer and front-end developer based in Melbourne, Australia. You can expect simple, smart and handsome websites designed and developed using modern responsive principles.

Responsive web design is built on a foundation of well considered HTML and CSS. Josh designs websites with a strong knowledge of how code performs online across numerous devices. This understanding is essential in providing websites that are simple for the sake of keeping speed fast, and costs low.
